El rincón colchonero es la web que todo buen aficionado del club Atletico de Madrid tiene que tener presente.En ella se pueden adquirir diversos productos del club para poder animar como el equipo se merece.

La web surge como trabajo de fin de evaluación, para mostrar los conocimientos adquiridos por un no tan joven estudiante de programación.

En la web podrás adquirir infinidad de productos originales. Todos los productos de nuestra web cuentan con el sello de calidad del club Atletico de Madrid.

Si encuentras en algún otro lugar un producto del equipo que nosotros no tenemos te aseguramos que no es original.

Si no te emocionas con los colores rojiblancos esta no es tu web. Socios del club acreditados tendrán envío gratuito.
